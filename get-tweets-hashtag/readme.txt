Get tweets by hashtag and display them

== Description ==

Retrieves Tweets from Twitter by hashtag and then displays them on your page or post by using a shortcode. Tweets are cached and the plugin
comes with a basic styling.

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place your Twitter Oauth settings in the Get Tweets by Hashtag Settings menu
4. Use shortcode on any page or post. [get-tweets hashtag="#yourhashtag"]
